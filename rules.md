# 成都麻将规则手册（2026年版）

## 一、基本设定

- **牌具**：仅使用筒（饼）、条（索）、万三种花色，每种花色1~9各4张，共108张牌。**不使用风牌（东南西北）与箭牌（中发白）**。
- **人数**：4人游戏。
- **初始手牌**：每人13张，庄家14张。
- **出牌顺序**：逆时针轮流出牌，摸一张、打一张，始终保持13张手牌（胡牌时为14张）。

## 二、核心规则

### 1. 打缺一门（定缺）

- 玩家**必须在胡牌前舍弃其中一种花色**（筒、条、万三选一），整局不得再保留或使用该花色牌。
- 若胡牌时仍持有三种花色，则视为“花猪”，需按满贯赔付其余三家。
- 定缺方式为 **打定张**：开局摸牌后立即公开所缺花色，不可更改。

### 2. 不可吃牌

- 成都麻将**只允许碰、杠，不允许吃牌**。
- “碰”：当任意玩家打出一张牌，你手中已有两张相同牌，可亮出组成刻子（三张相同）。
- “杠”：分三种类型（见下文“刮风下雨”）。

### 3. 血战到底

- **一家胡牌后，牌局继续**，其余未胡玩家继续打牌，直到：
  - 三人胡牌；或
  - 牌墙摸完且剩余两人及以上未胡（流局）。

## 三、胡牌规则

### 1. 基本胡牌结构

- 标准胡牌型为 **“3+3+3+3+2”**，即四组顺子或刻子 + 一对将牌（对子）。
- 必须满足“缺一门”条件才能胡牌。

### 2. 特殊胡牌型与番数（计分倍数）

| 胡牌类型 | 番数 | 说明 |
|----------|------|------|
| 平胡 | 1番 | 缺一门，四组顺/刻 + 一对将 |
| 对对胡 | 2番 | 全部为刻子（无顺子） |
| 清一色 | 3番 | 全副牌为同一花色（如全万） |
| 七对 | 3番 | 七组对子（不遵循3+3+3+3+2结构） |
| 将对 | 3番 | 全为2、5、8组成的对子或刻子 |
| 龙七对 | 4番起 | 七对中有两对为四张相同（未杠） |
| 清对 | 4番 | 清一色 + 对对胡 |
| 天胡 / 地胡 | 8番（通常不计分） | 开局即胡（天胡）或首张出牌即胡（地胡） |

### 3. 胡牌方式

- **自摸**：自己摸到所需胡牌，赢其余所有未胡玩家。
- **点炮**：他人打出你需要的牌，仅点炮者赔付。
- **一炮多响**：若一张牌同时被多人胡，则**所有听牌者均可胡牌**，点炮者赔付全部。
- **抢杠胡**：他人补杠时打出的牌正好是你所听之牌，可胡，且杠分作废。

## 四、杠牌机制（“刮风下雨”）

杠牌不仅改变手牌结构，还即时结算分数：

| 杠类型 | 触发条件 | 收益对象 | 收益倍数 |
|--------|----------|----------|----------|
| 直杠（明杠） | 手中有三张，他人打出第四张 | 引杠者 | 1倍 |
| 面下杠（补杠） | 已碰三张，自摸第四张 | 其他未胡玩家 | 1倍 |
| 暗杠 | 自摸四张相同牌（未碰） | 其他未胡玩家 | 2倍 |

- **杠上花**：杠后自摸胡牌，**额外加1~2番**。
- **杠上炮**：杠后打出的牌被他人胡，**加1番**。
- **流局时未听牌**：需**退还所有杠牌所得**。

## 五、流局处理（查叫与查花猪）

当牌局结束无人胡牌（或血战模式下剩两人未胡）时：

1. **查花猪**：  
   - 手持三门花色者为“花猪”，**赔付非花猪玩家16倍基础分**。
2. **查大叫**：  
   - 未听牌者（非花猪）需按“最大可能胡牌番数”赔付已听牌者，并退还杠分。

## 六、其他重要规则

- **庄家规则**：通常由上局首位胡牌者担任；若一炮多响，则点炮者为庄。

## 七、术语速查

| 术语 | 含义 |
|------|------|
| 花猪 | 未打缺一门者 |
| 听牌（下叫） | 只差一张即可胡牌 |
| 刮风 | 明杠 |
| 下雨 | 暗杠 |
| 血战 | 一人胡后继续打 |
| 查叫 | 流局时检查是否听牌 |
